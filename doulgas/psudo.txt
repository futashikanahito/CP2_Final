Start the app with real-time messaging support.
load_json(path): Read data from a file. Return nothing if the file doesn't exist.
save_json(path, data): Make any needed folders, then save data to a file.
get_user(username): Get a user's data by their username.
is_server_owner(username, server_id): Check if a user owns a server.
is_server_member(username, server_id): Check if a user is in a server.
Frontend /: Send the user the main webpage.
User Management
/user/create: Make a new user file if one doesn't already exist.
/user/servers/: Return the list of servers a user is in.
Server Management
/server/create: Make a new server with a general channel and link it to the creator's account.
/server/delete: Confirm ownership, delete the server, and unlink it from the owner's account.
/server/adduser: Confirm ownership, add a new member to the server, and link the server to their account.
/server/channels/<server_id>: List all channels in a server.
Messaging
/message/send: Confirm membership, save the message to the channel, and send it to everyone in the channel instantly.
/message/get/<server_id>/: Return the message history for a channel.
Real-Time Socket Events
join: Add a user to a live channel room.
leave: Remove a user from a live channel room.
Run App Start the server on port 5000 with debugging on.
