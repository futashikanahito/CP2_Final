
# WorkFlow - Business Optimization App
***
The all in one workspace for all the business needs ranging from planning a new website, to getting a new building up and running, WorkFlow can help with it all! From general features like notifications, notepads, checklists, all the way up to group chats/channels and even video calls, making sure the WorkFlow can fit your every need and more. Made by a small group of junior developers for school, this project is free to use and manipulate however you see fit, and however it best fits your company!

***

## Details on Project features
***
#### Main Features Include:
- A functioning accounts system for each user
- A complex dashboard that fits all your organizing needs
- Calendars for planning any event happening
- Group chats for communication between coworkers
- Video Calls for all the meetings you would ever need
- A notification system to stay up to date on all the project needs
- A checklist to make sure you get all those tasks done
- A notepad for general notes during an important meeting or many other uses
